# The PaaStel.io Book

[_The PaaStel.io Book_](https://book.paastel.io/) serves as PaaStel primary
documentation and tutorial resource.

## How to Build from Source

* Install [`mdbook`](https://github.com/rust-lang/mdBook)

```sh
cargo install mdbook
```
* Run `build`

```sh
mdbook build
```
